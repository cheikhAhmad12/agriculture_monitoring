# agriculture_monitoring

Raspberry Pi side implementation for the drone (Navio2 + camera) and irrigation controller.

## Raspberry Pi code
- Location: `raspberry/`
- Entrypoint: `python -m raspberry.main`
- Dependencies: `pip install -r raspberry/requirements.txt`
- Configuration: edit `raspberry/config/default_config.yaml`

See `raspberry/README.md` for role-specific instructions and `raspberry/DEPLOYMENT.md` for Pi-side setup.
